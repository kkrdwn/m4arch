# General Packages Needed for Arch Linux Setup

This document lists general packages required for the Arch Linux setup that are not specific to the hardware.
The user should populate this list with their preferred applications, utilities, and configuration tools.

---

## Example/Placeholder Packages

* `hyprland` (Window manager/compositor)
* `waybar` (Status bar)
* `kitty` (Terminal emulator)
* `neovim`/ `vscode` (Text editor)
* `dunst` (Notification daemon)

---

## Instructions for User

Please add all general software packages you typically install on your Arch Linux setup here. This file will eventually be used by the Rust application to manage these installations.
